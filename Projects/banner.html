Interactive Banner

The purpose of this project was to create an interactive web banner that responds to user actions. I chose to create an interactive banner that follows a skier going down a mountain with the user in the skier's point of view. When the user clicks the red button, they will advance the banner forward and trigger specific animations. There are four scenes that the user can interact with. When clicking “Let's Ski,” the skier will push off using their poles through a couple of trees. “Avoid!” maneuvers the skier around a rock. “Apres Ski” continues the skier down the mountain where they find a beverage. Finally, “Ride Lift” will restart the interaction so the user can re-cycle through.
